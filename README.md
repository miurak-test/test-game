# 恋も人生もダイス次第 ～人生、全部賭ける。～

コージー生活シムのすごろくゲーム。

## 遊び方

1. 名前と性別を選んで冒険に出発
2. サイコロを振ってマスに進む
3. イベントが発生 — 選択次第で幸福度が変動
4. 春夏秋冬を巡って、幸福度スコアを競おう

## 特徴

- **ゆらぎ幸福論**: 良いことの裏に疲労、悪いことに学びがある
- **分岐ルート**: 農業・お店・村づくりの3つの道
- **恋愛要素**: NPC との交流で恋心が芽生えるかも？
- **称号システム**: プレイスタイルに応じた称号を獲得
- **年鑑ミニ**: プレイの思い出を振り返ろう

## 開発

```bash
bun install
bun run dev      # 開発サーバー起動
bun run build    # 本番ビルド
bun run test     # テスト実行
```

## 技術スタック

- Phaser 3 + TypeScript
